Personal Finance Manager
------------------------------

Description of project:

Personal Finance Manager is a Python-based application that helps users manage their personal finances. 
Users can register, log in, record income and expenses, and view their transaction history. 
The system stores data using SQLite and allows users to track their financial activities efficiently.
----------------------------------
Features:
 1.User Registration
 2.User Login Authentication
 3.Add Income
 4.Add Expense
 5.View Transaction History
 6.Balance Calculation
 7.Data stored using SQLite database
-----------------------------------

Technologies Used

1. Python
2. SQLite Database
3. Python unittest (for testing)
